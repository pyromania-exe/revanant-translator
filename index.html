Revanant Code – Rules

    
Word Order

    
  • OVS (Object–Verb–Subject). Example: Book read I.

    
Pronouns

    
  • ra → I
  • vi → You
  • tor → We
  • zen → They/He/She/It

    
Tense & Negation

    
  • Past: -eth
  • Future: va-
  • Negation: x-

    
Modifiers

    
  • Adjectives follow nouns with vowel shift (a→u, e→i, i→o, o→a, u→e).

    
Pluralization

    
  • Double first consonant: dog → ddog.

    
Markers

    
  • ¤ sentence divider
  • ~ emphasis
  • : possession
  • // uncertainty

    
Examples

    
    
      
  • Gate va-guard ra¤ → I will guard the gate.
    • 
      
  • Truth x-speaketh zen¤ → They did not speak truth.
    • 
      
  • Path walk vi¤ → You walk the path.
    • 
      
  • Banner radu claim tor¤ → We claim the red banner.
